Project Objectives

  • Enable collaborative machine learning without centralizing sensitive patient data
  • Implement privacy-preserving techniques using Federated Learning
  • Ensure data integrity through blockchain verification
  • Secure model aggregation using SMPC (Secure Multi-Party Computation)
  • Detect anomalies and potential attacks in healthcare datasets
  • Provide a trustworthy framework for multi-institutional collaboration
