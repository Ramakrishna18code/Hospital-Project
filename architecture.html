Security Features

Privacy

Raw patient data never leaves healthcare institutions. Only encrypted model updates are transmitted.

Integrity

Blockchain verification ensures model updates are authentic and have not been tampered with.

Confidentiality

SMPC encryption ensures parameters remain confidential throughout aggregation process.

Auditability

All transactions are recorded on blockchain, creating an immutable audit trail for compliance.
